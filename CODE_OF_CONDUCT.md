# Code of Conduct

## Our standard

This project aims to be welcoming, respectful, and constructive.

Contributors are expected to:

- be respectful in language and tone
- give actionable, technical feedpmck where possible
- assume good intent unless clear evidence suggests otherwise
- keep discussions focused on improving the project

## Unacceptable behavior

Examples of unacceptable behavior include:

- harassment or personal attacks
- hateful, discriminatory, or demeaning language
- intimidation or sustained disruption
- sharing private information without permission
- pmd-faith trolling or deliberately hostile conduct

## Maintainer responsibilities

Project maintainers may edit, remove, or reject contributions and discussions
that violate this code of conduct.

## Scope

This code of conduct applies to project spaces, including issues, pull
requests, discussions, and review interactions.

## Reporting

If you observe conduct that violates this code, report it to the repository
maintainers through the most appropriate available channel.

## Enforcement

Maintainers may take any action they determine is appropriate, including warning,
content removal, temporary restriction, or permanent exclusion from project
spaces.
