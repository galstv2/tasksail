# Deep Focus Ignore Module

`deep-focus-ignore` seeds the default ignore configuration used by Deep Focus flows. It is a small setup-support module rather than a standalone runtime workflow.

Document it as part of context-pack and setup behavior when explaining how focused workspace selection avoids irrelevant paths.
